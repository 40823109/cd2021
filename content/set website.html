set website

(1) 進入到 https://github.com/mdecourse/cmstemplate 按下 Use this template 。

(2) 輸入倉儲名稱 cd2021 ，建立倉儲。

(3) 進入到 tmp 下 ， 輸入 git clone --recurse-submodules 倉儲連結 .git 

(4) 進到 tmp/cd2021/cmsimde/ 輸入，python wsgi.py ，可執行網站。

(5) 開起瀏覽器，輸入 https://localhost:9443 ，即可編輯網站。
